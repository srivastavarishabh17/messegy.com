HERO VIDEO SETUP
================

Add your video file here:
- File name: hero-video.mp4
- Recommended: 1920x1080 resolution, H.264 codec
- Duration: 10-30 seconds (looping)
- File size: Under 5MB for fast loading

VIDEO SOURCES (Free):
====================
1. Pexels.com - Search: "abstract technology", "dark gradient", "business communication"
2. Pixabay.com - Search: "digital network", "technology background"
3. Coverr.co - Free video backgrounds

RECOMMENDED VIDEO TYPES:
=======================
- Abstract tech animation with flowing particles
- Dark gradient loop (dark blue/purple colors)
- Subtle animated chat bubbles
- Professional office/team using devices
- Network connections visualization

If no video is added, the hero will show:
1. Video (if exists)
2. Fallback image at /images/hero-bg.jpg (if exists)
3. Default dark blue gradient (always shows)
